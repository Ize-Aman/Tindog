# TinDog 🐶

TinDog is a simple responsive landing page inspired by Tinder, but for dogs.
This project was created as part of a web development course by Dr. Angela Yu.

The goal of the project was to practice building responsive layouts using Bootstrap and understand how modern UI components work together.

---

## 🚀 Features

- Fully responsive landing page
- Built using Bootstrap components
- Clean and modern layout
- Sections for features, pricing, testimonials, and download links

---

## 🛠️ Built With

- HTML5
- CSS3
- Bootstrap

---

## 📚 What I Learned

Through this project I practiced:

- Using Bootstrap’s grid system
- Creating responsive layouts
- Working with pre-built UI components (cards, buttons, navbar, etc.)
- Structuring a simple landing page

---

## 🎯 Purpose of the Project

This project was built for learning purposes while following a web development course.
It helped me understand how Bootstrap simplifies responsive design and speeds up front-end development.

---

## 📂 How to Run

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
